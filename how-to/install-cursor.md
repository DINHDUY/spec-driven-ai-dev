# How to Download and Install Cursor

## For Mac

1. **Download Cursor**
   - Go to https://cursor.com
   - Click the "Download for Mac" button

2. **Install Cursor**
   - Open your Downloads folder and find the downloaded file (Cursor.dmg)
   - Double-click the DMG file to open it
   - Drag the "Cursor" app to your Applications folder

3. **Open Cursor**
   - Go to your Applications folder
   - Double-click "Cursor" to launch it
   - If you see a security warning, go to System Preferences > Security & Privacy and click "Open Anyway"

## For Windows

1. **Download Cursor**
   - Go to https://cursor.com
   - Click the "Download for Windows" button

2. **Install Cursor**
   - Open your Downloads folder and find the downloaded file (cursor-setup.exe)
   - Double-click the installer file
   - Follow the setup wizard:
     - Click "Install"
     - Wait for the installation to complete

3. **Open Cursor**
   - Cursor will launch automatically after installation, or
   - Find it in your Start Menu and click to open

---

**That's it!** Cursor is now installed and ready to use on your computer.
