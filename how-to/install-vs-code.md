# How-to Download and Install VS Code

## For Mac

1. **Download VS Code**
   - Go to https://code.visualstudio.com
   - Click the big "Download for Mac" button

2. **Install VS Code**
   - Open your Downloads folder and find the downloaded file (VSCode-darwin-universal.zip)
   - Double-click the zip file to extract it
   - Drag the "Visual Studio Code" app to your Applications folder

3. **Open VS Code**
   - Go to your Applications folder
   - Double-click "Visual Studio Code" to launch it

## For Windows

1. **Download VS Code**
   - Go to https://code.visualstudio.com
   - Click the big "Download for Windows" button

2. **Install VS Code**
   - Open your Downloads folder and find the downloaded file (VSCodeUserSetup-x64.exe)
   - Double-click the installer file
   - Follow the setup wizard:
     - Accept the license agreement
     - Click "Next" through the steps
     - Click "Install"

3. **Open VS Code**
   - VS Code will launch automatically after installation, or
   - Find it in your Start Menu and click to open

---

**That's it!** VS Code is now installed and ready to use on your computer.
